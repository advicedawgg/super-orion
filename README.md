# Super Orion ⭐
*A Totally Legit Adventure*

A silly, charming 2D platformer made for Orion. No installs, no internet needed.

## How to play
**Just double-click `index.html`** — it opens in your browser and runs instantly.

| Key | Action |
|---|---|
| ← → or A / D | Move |
| SPACE or ↑ / W | Jump (hold for higher jumps!) |
| ↓ or S | Duck |
| P | Pause (with very helpful "tips") |
| M | Music on/off |
| R | Restart level |
| ENTER | Start / continue |

On a tablet or phone, on-screen buttons appear automatically.

## The game
- **5 levels:** Grassy Greens → Cookie Caves → Sky Snack Summit → Soda Springs → Star Fortress
- Collect **gold stars** (Orion = constellation, get it?), bonk **Grumblins** and **Flapjacks** by jumping on them, bounce on springs, break bricks, bump `?` blocks, and touch the star flag to win.
- Kid-friendly: 3 hearts per life, 5 lives, infinite continues, and the Game Over screen just suggests a snack break.

## 🤫 Parent's Secret Easter Egg Guide
*(Don't tell Orion — let him discover these! Or drop hints...)*

| Easter egg | How to trigger |
|---|---|
| 💃 **Idle dance** | Stand completely still for ~6 seconds. Orion gets bored and starts dancing with music notes. |
| 🌈 **Rainbow Mode** | Type the letters `O-R-I-O-N` on the keyboard any time. Rainbow trail! Type it again to turn off. |
| 🧠 **Big Brain Mode** | The Konami code: ↑ ↑ ↓ ↓ ← → ← → B A. Giant head. |
| 🌮 **Taco Power** | Hidden in special `?` blocks — super speed! Also try *typing* `taco`... |
| 🤭 **The Toot** | Press duck (↓) 5 times quickly. "Oops. Beans for breakfast." |
| 🧱 **The Wall** | Push against a wall for a few seconds. The wall wins. |
| 🚀 **Maximum Boing** | Jump 12 times in a row as fast as possible. Confetti! |
| 🟢 **Secret pipe** | Levels 1 and 4 each have a pipe that *sparkles*. Stand on it and press ↓ — secret star stash! |
| 👻 **Invisible block** | Somewhere in level 1, jump under empty air near a lone brick... free 1-UP. |
| 💥 **Combo bonks** | Stomp 2–3 enemies without touching the ground: DOUBLE BONK! / TRIPLE BONK!!! |
| 🏆 **Star Champion** | Collect every star in a level before the flag for a bonus. |
| 😴 **Pause tips** | Pause the game repeatedly. The tips are... very helpful. |

The win screen tracks stats like "Enemies bonked", "Gravity lessons learned" (deaths), and "Mysterious toots."
